## Customisation

### Customisation properties
You can use the `theme` property to customise the component. See all available theme variables in the table below.

| Theme variable                                   | Description                                         |
| ------------------------------------------------ | --------------------------------------------------- |
| `--dropdown-background-default`                  | Background color of dropdown                        |
| `--dropdown-background-disabled`                 | Background color of disabled dropdown               |
| `--dropdown-border-color-default`                | Border color of dropdown                            |
| `--dropdown-border-color-hover`                  | Border color of hovered dropdown                    |
| `--dropdown-border-color-hover-invalid`          | Border color of hovered and invalid dropdown        |
| `--dropdown-border-color-focus`                  | Border color of focused dropdown                    |
| `--dropdown-border-color-invalid`                | Border color of invalid dropdown                    |
| `--dropdown-border-color-disabled`               | Border color of disabled dropdown                   |
| `--dropdown-color-default`                       | Dropdown color                                      |
| `--dropdown-color-disabled`                      | Disabled dropdown color                             |
| `--focus-outline-color`                          | Focus outline color                                 |
| `--helper-color-default`                         | Helper text color                                   |
| `--helper-color-invalid`                         | Helper text color when dropdown state is invalid    |
| `--menu-divider-color`                           | Menu divider color                                  |
| `--menu-item-background-default`                 | Background color of menu item                       |
| `--menu-item-background-hover`                   | Background color of hovered menu item               |
| `--menu-item-background-selected`                | Background color of selected menu item              |
| `--menu-item-background-selected-hover`          | Background color of selected and hovered menu item  |
| `--menu-item-background-disabled`                | Background color of disabled menu item              |
| `--menu-item-color-default`                      | Color of menu item                                  |
| `--menu-item-color-hover`                        | Color of hovered menu item                          |
| `--menu-item-color-selected`                     | Color of selected menu item                         |
| `--menu-item-color-selected-hover`               | Color of selected and hovered menu item             |
| `--menu-item-color-disabled`                     | Color of disabled menu item                         |
| `--menu-item-icon-color-selected`                | Color of selected menu item icon                    |
| `--menu-item-icon-color-disabled`                | Color of disabled menu item icon                    |
| `--multiselect-checkbox-background-selected`     | Background color of selected multiselect checkbox   |
| `--multiselect-checkbox-background-disabled`     | Background color of disabled multiselect checkbox   |
| `--multiselect-checkbox-border-default`          | Color of multiselect checkbox border                |
| `--multiselect-checkbox-border-hover`            | Color of hovered multiselect checkbox border        |
| `--multiselect-checkbox-border-disabled`         | Color of disabled multiselect checkbox border       |
| `--multiselect-checkbox-color-default`           | Color of multiselect checkbox                       |
| `--multiselect-checkbox-color-selected`          | Color of selected multiselect checkbox              |
| `--multiselect-checkbox-color-selected-disabled` | Color of selected and disabled multiselect checkbox |
| `--placeholder-color`                            | Placeholder color                                   |
