## Customisation

### Customisation properties
You can use the `theme` property to customise the Accordion component. See all available theme variables in the table below.

| Theme variable                           | Description                                     |
| ---------------------------------------- | ----------------------------------------------- |
| <code>--background-color</code>          | Background color of the accordion.              |
| <code>--button-border-color-hover</code> | Background color of the accordion when hovered. |
| <code>--button-size</code>               | Size of the angle icon.                         |
| <code>--content-font-color</code>        | Font colour of the accordion content.           |
| <code>--content-font-size</code>         | Font size of the accordion content.             |
| <code>--content-line-height</code>       | Line height of the accordion content.           |
| <code>--header-font-color</code>         | Font colour of the accordion header.            |
| <code>--header-font-size</code>          | Font size of the accodion header.               |
| <code>--header-line-height</code>        | Line height of the accordion header.            |
| <code>--padding-horizontal</code>        | Left and right padding of the accordion.        |
| <code>--padding-vertical</code>          | Top and bottom padding of the accordion.        |
