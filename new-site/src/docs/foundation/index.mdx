# Foundation overview

<LeadParagraph>
  Foundation is a collection of basic guidelines and principles for starting to design and develop user experiences for the City of Helsinki.
</LeadParagraph>

Helsinki Design System foundations describe how the <Link size="M" href="https://brand.hel.fi/en/" external>Helsinki Visual Identity</Link> is applied to digital world. Foundation varies from detailed practical guides to broader principles that needs to be addressed when designing and developing a service.

**Design tokens** are the smallest building blocks of the design system. They are used across all Helsinki products to ensure matching visual identity.

**Visual assets** define the common set of graphical elements used in digital services.

**Guidelines** give answers and suggestions to common situations we come across in digital products. They describe the Helsinki way of solving these issues.
