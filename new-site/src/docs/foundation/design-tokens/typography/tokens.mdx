## Tokens

<Notification label="Consider using helper classes" className="siteNotification">
  Instead of using tokens directly to style your typography, consider using <Link size="S" href="https://github.com/City-of-Helsinki/helsinki-design-system/tree/master/packages/core/src/utils/helpers.css" external>helper classes provided in the Core package</Link>. In these classes, you do not need to worry about matching the correct font weight, line height and letter spacing are matched to the chosen font size.
</Notification>
