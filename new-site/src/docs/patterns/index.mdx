# Patterns overview

<LeadParagraph>
  Patterns a collection of design principles and solutions. They consist of guidelines and component examples of how to address common user objectives.
</LeadParagraph>

Patterns describe how [HDS components](/components/) are used together to build solutions that are coherent across the whole city.
