    Dropdown

    StableAccessible

    A dropdown offers a user a list of options, of which one or multiple can be selected. Dropdowns are often used as part of forms and filters. HDS offers two types of dropdowns, a Select and a Combobox component.

    Dropdowns are deprecated

    Combobox is deprecated and will be removed in the next major release.

    Select is being redesigned and will have different props in the next major release.

    Customisation properties

    You can use the theme property to customise the component. See all available theme variables in the table below.

    Theme variableDescription
    --dropdown-background-defaultBackground color of dropdown
    --dropdown-background-disabledBackground color of disabled dropdown
    --dropdown-border-color-defaultBorder color of dropdown
    --dropdown-border-color-hoverBorder color of hovered dropdown
    --dropdown-border-color-hover-invalidBorder color of hovered and invalid dropdown
    --dropdown-border-color-focusBorder color of focused dropdown
    --dropdown-border-color-invalidBorder color of invalid dropdown
    --dropdown-border-color-disabledBorder color of disabled dropdown
    --dropdown-color-defaultDropdown color
    --dropdown-color-disabledDisabled dropdown color
    --focus-outline-colorFocus outline color
    --helper-color-defaultHelper text color
    --helper-color-invalidHelper text color when dropdown state is invalid
    --menu-divider-colorMenu divider color
    --menu-item-background-defaultBackground color of menu item
    --menu-item-background-hoverBackground color of hovered menu item
    --menu-item-background-selectedBackground color of selected menu item
    --menu-item-background-selected-hoverBackground color of selected and hovered menu item
    --menu-item-background-disabledBackground color of disabled menu item
    --menu-item-color-defaultColor of menu item
    --menu-item-color-hoverColor of hovered menu item
    --menu-item-color-selectedColor of selected menu item
    --menu-item-color-selected-hoverColor of selected and hovered menu item
    --menu-item-color-disabledColor of disabled menu item
    --menu-item-icon-color-selectedColor of selected menu item icon
    --menu-item-icon-color-disabledColor of disabled menu item icon
    --multiselect-checkbox-background-selectedBackground color of selected multiselect checkbox
    --multiselect-checkbox-background-disabledBackground color of disabled multiselect checkbox
    --multiselect-checkbox-border-defaultColor of multiselect checkbox border

    --multiselect-checkbox-border-disabledColor of disabled multiselect checkbox border
    --multiselect-checkbox-color-defaultColor of multiselect checkbox
    --multiselect-checkbox-color-selectedColor of selected multiselect checkbox
    --multiselect-checkbox-color-selected-disabledColor of selected and disabled multiselect checkbox
    --placeholder-colorPlaceholder color

      const options = [{ label: 'Plutonium' }, { label: 'Americium' }, { label: 'Copernicium' }, { label: 'Berkelium' }];
