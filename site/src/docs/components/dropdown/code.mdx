### Properties
Note! You can find the full list of properties in the <ExternalLink href="/storybook/react/?path=/docs/components-dropdowns-combobox--default">React Storybook.</ExternalLink>

Also, note that this component is an input. All features supported by the HDS TextInput are also supported by this component. See <InternalLink href="/components/text-input">TextInput documentation page</InternalLink> for more information.

| Property      | Description                                                       | Values         | Default value |
| ------------- | ----------------------------------------------------------------- | -------------- | ------------- |
| `clearable`   | If set to true, the clear selections button is enabled.           | `boolean`      | false         |
| `disabled`    | If set to true, the dropdown is disabled.                         | `boolean`      | false         |
| `helper`      | Helper text to be displayed below the input.                      | `string`       | -             |
| `error`       | Error text to be displayed below the input.                       | `string`       | -             |
| `icon`        | An icon to be shown in the beginning of the input.                | `ReactNode`    | -             |
| `label`       | A text label for the input.                                       | `string`       | -             |
| `options`     | The array of options shown in the input.                          | `OptionType[]` | -             |
| `placeholder` | Short hint displayed in the input before the user enters a value. | `string`       | -             |
| `required`    | If set to true, the input is marked as required.                  | `boolean`      | false         |
| `multiselect` | If set to true, the user can select multiple values.              | `boolean`      | false         |
