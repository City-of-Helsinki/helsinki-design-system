### Properties
Note! You can find the full list of properties in the <ExternalLink href="/storybook/react/?path=/story/components-dateinput--default">React Storybook.</ExternalLink>

Also, note that this component is an input. All features supported by the HDS TextInput are also supported by this component. See [TextInput documentation page](/components/text-input) for more information.

| Property                                         | Description                                                    | Values                                       | Default value                                   |
| ------------------------------------------------ | -------------------------------------------------------------- | -------------------------------------------- | ----------------------------------------------- |
| `helperText`                                     | Helper text to be displayed below the input.                   | `string`                                     | -                                               |
| `errorText`                                      | Error text to be displayed below the input.                    | `string`                                     | -                                               |
| `successText`                                    | Success text to be displayed below the input.                  | `string`                                     | -                                               |
| `infoText`                                       | Info text to be displayed below the input.                     | `string`                                     | -                                               |
| `initialMonth`                                   | The initial month open in calendar.                            | `Date`                                       | -                                               |
| `label`                                          | A text label for the input.                                    | `string`                                     | -                                               |
| `onChange`                                       | Callback fired when the state is changed.                      | `event`                                      | -                                               |
| `language`                                       | Language of the date input.                                    | `string` (fi, en, sv)                        | en                                              |
| `minDate`                                        | Minimum date to show in the datepicker calendar.               | `Date`                                       | Start of the month one year before current date |
| `maxDate`                                        | Maximum date to show in the datepicker calendar.               | `Date`                                       | End of the month one year from current date.    |                                                 |
| `isDateDisabledBy`                               | Disables date(s) based on conditional function.                | `function`                                   | -                                               |
| `initialMonth`                                   | The initial month open in calendar.                            | `Date`                                       | -                                               |
| `disableDatePicker`                              | If set to true, the date picker button is hidden.              | `boolean`                                    | false                                           |
| `disableConfirmation`                            | If set to true, the date picker does not require confirmation. | `boolean`                                    | false                                           |
