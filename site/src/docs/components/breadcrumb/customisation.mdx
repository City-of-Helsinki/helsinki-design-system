## Customisation

### Customisation properties

In Core version, you can either use the `style` or `class` attributes in the HTML to customise the component.

In React version, you can use the `theme` property to customise the component.

See all available theme variables in the table below.

| Theme variable                | Description                                                                          |
| ----------------------------- | ------------------------------------------------------------------------------------ |
| `--horizontal-margin-small`   | Horizontal margin on small screens (< 768px). Default is `--spacing-layout-2-xs`     |
| `--horizontal-margin-medium`  | Horizontal margin on medium screens (>=768px). Default is `--spacing-layout-xs`      |
| `--horizontal-margin-large`   | Horizontal margin on large screens (>= 992px). Default is `--spacing-layout-xs`      |
| `--horizontal-margin-x-large` | Horizontal margin on xtra large screens (>= 1248px). Default is `--spacing-layout-s` |

### Customisation example

<Playground scope={{ Breadcrumb }}>

```jsx
import { Breadcrumb } from 'hds-react';

() => (
<Breadcrumb
  aria-label="Breadcrumb"
  list={[
    { title: 'Front page', path: '/' },
    { title: 'Health and social services', path: '/path' },
    { title: 'Senior services', path: '/path/2ndLevelPath' },
    { title: 'Informal care', path: '/path/2ndLevelPath/3rdLevelPath' },
    { title: 'Care options', path: null },
  ]}
  theme={{
    '--horizontal-margin-small': 'var(--spacing-layout-s)',
    '--horizontal-margin-medium': 'var(--spacing-layout-m)',
    '--horizontal-margin-large': 'var(--spacing-layout-l)',
    '--horizontal-margin-x-large': 'var(--spacing-layout-xl)',
  }}
/>)
```
