## Accessibility

The HDS breadcrumb component is a native HTML list with native HTML links.

### Pay attention to

- The last element should have the attribute `aria-current="true"` if it is the current page. Automatically added in React.
- Nav-element must have an `aria-label`. Automatically created in React.
