### Properties
Note! You can find the full list of properties in the <Link openInNewTab openInNewTabAriaLabel="Opens in a new tab" href="/storybook/react/?path=/story/components-logo">React Storybook.</Link>


| Property                 | Description                                                                                | Values                 | Default value |
| ------------------------ | ------------------------------------------------------------------------------------------ | ---------------------- | ------------- |
| `logoLanguage`           | Language of the logo.                                                                      | `"fi"`, `"sv"`, `"en"` | `"fi"`        |
| `menuToggleAriaLabel`    | The aria-label for the mobile menu toggle button.                                          | `string`               | -             |
| `onTitleClick`           | Callback fired when the title or logo is clicked.                                          | `function`             | -             |
| `skipTo`                 | ID of the element to jump to when the "Skip to content" accessibility shortcut is clicked. | `string`               | -             |
| `skipToContentAriaLabel` | Aria-label for the "skip to content" accessibility shortcut.                               | `string`               | -             |
| `skipToContentLabel`     | Text for the "skip to content" accessibility shortcut.                                     | `ReactNode`            | -             |
| `title`                  | The title of the service shown next to the logo.                                           | `ReactNode`            | -             |
| `titleAriaLabel`         | The aria-label for the title describing the logo and service to screen reader users.       | `ReactNode`            | -             |
| `titleUrl`               | URL to navigate to when the logo or title is clicked.                                      | `string`               | -             |
[Table 1:Navigation properties]|

| Property              | Description                                                                                                       | Values                                     | Default value |
| --------------------- | ----------------------------------------------------------------------------------------------------------------- | ------------------------------------------ | ------------- |
| `variant`             | Defines where the navigation row will be displayed.                                                               | `"default"`, `"inline"`,                   | `"default"`   |
| `ariaLabel`           | The aria-label for helping screen reader users to distinguish navigation row from other navigational components   | `string`                                   | -             |

| `label`   | The label for the item. Optionally, children can be passed.                                          | `ReactNode`                                   | -             |
| `variant` | Defines the button variant in mobile view. Intended to be used within the `NavigationUser` component | `"primary"`, `"secondary"`, `"supplementary"` | -             |
