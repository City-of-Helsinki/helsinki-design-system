## Customisation

### Customisation properties
In Core version, you can use button theme classes `hds-button--theme-black` and `hds-button--theme-coat` for different colour schemes.

In React version, you can use the `theme` property to customise the component.

See all available theme variables in the table below.


| Theme variable     | Description                              |
| ------------------ | ---------------------------------------- |
| <code>coat</code>  | Background color of the button is coat.  |
| <code>black</code> | Background color of the button is black. |


### Customisation example

<Playground>


  ```jsx
  <Button theme="coat" iconRight={<IconArrowRight />}>
    Customised button
  </Button>
  ```

  ```html
  <button type="button" class="hds-button hds-button--primary hds-button--theme-coat">
    <span class="hds-button__label">Customised button</span>
    <span aria-hidden="true" class="hds-icon hds-icon--arrow-right"></span>
  </button>
  ```
