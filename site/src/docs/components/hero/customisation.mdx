## Customisation

### Customisation properties

In Core version, you can either use the `style` or `class`Â attributes in the HTML to customise the component.

In React version, you can use the `theme` property to customise the component.

See all available theme variables in the table below.

| Theme variable                 | Description                                                                                          |
| ------------------------------ | ---------------------------------------------------------------------------------------------------- |
| `--arrow-icon-color`           | Arrow icon color. Default is the current text color (`--color`).                                                   |
| `--background-color`           | Background color. Default is `--color-white`                                                         |
| `--color`                      | Text color. Default is `--color-black-90`                                                            |
| `--image-position`             | How image is aligned if it does not fit the container. Default is `bottom right`                     |
| `--koros-color`                | Color of the koros. Defaults to `--background-color`                                                 |
| `--koros-height`               | Height of the koros. In React this defaults to the height of the koros shape                         |
| `--diagonal-koros-position`    | How is angled koros positioned. This is applied to the css property `right`.Default is `right: 45%`. |
| `--horizontal-padding-small`   | Horizontal padding on small screens (< 768px). Default is `--spacing-layout-2-xs`                    |
| `--horizontal-padding-medium`  | Horizontal padding on medium screens (>=768px). Default is `--spacing-layout-xs`                     |
| `--horizontal-padding-large`   | Horizontal padding on large screens (>= 992px). Default is `--spacing-layout-xs`                     |
| `--horizontal-padding-x-large` | Horizontal padding on xtra large screens (>= 1248px). Default is `--spacing-layout-xs`               |

In React version, you can use the `koros` property to customise the Koros component.

You can use the same properties as the <Link href="/components/koros/customisation">Koros component</Link> and also an additional property `hide`.

| Koros property | Description                          |
| -------------- | ------------------------------------ |
| `hide`         | If set to `true`, koros is not shown |

### Customisation example

<Playground scope={{ Hero, Button }}>

```jsx
import { Hero } from 'hds-react';

() => (
<Hero
  imageSrc="https://hds.hel.fi/images/homepage/amos58.jpg"
  variant="imageBottom"
  theme={{
    '--background-color': '#fff',
    '--arrow-icon-color': 'var(--color-brick)',
  }}
  koros={{ style: 'pulse', dense: true }}
  showArrowIcon
>
  <Hero.Title>Hero title</Hero.Title>
  <Hero.Text>
    This is an introduction text. The length is circa two sentences. The suitable total length of the title and
    introduction is around 230 characters, including spaces. The intro ends with a period.
  </Hero.Text>
  <Button
    variant="secondary"
    role="link"
    style={{ '--background-color': '#000', '--color': '#fff', '--border-color': '#000' }}
  >
    Call to action
  </Button>
</Hero>)
```
