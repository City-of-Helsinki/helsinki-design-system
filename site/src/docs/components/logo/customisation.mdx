## Customisation

<Notification type="alert" label="Logo customisation works differently!" className="siteNotification">
  Logo is an older HDS component and therefore its customisation works slightly differently. Instead of the 
  <code>theme</code> property, you need to use the <code>style</code> property. This will be fixed in a future update.
</Notification>

### Customisation properties

You can either use the `style` or `className` properties to customise the component.

### Customisation example

<Playground scope={{ Logo, LogoSize, logoFi }}>

```jsx

import { Logo, LogoSize, logoFi } from 'hds-react';

<>
  <Logo src={logoFi} size={LogoSize.Small} style={{ color: 'var(--color-bus)', marginRight: 'var(--spacing-xs)' }} alt="Helsingin kaupunki" />
  <Logo src={logoFi} size={LogoSize.Medium} style={{ color: 'var(--color-tram)', marginRight: 'var(--spacing-xs)' }} alt="Helsingin kaupunki" />
  <Logo src={logoFi} size={LogoSize.Large} style={{ color: 'var(--color-metro)' }} alt="Helsingin kaupunki" />
</>
```

</Playground>
