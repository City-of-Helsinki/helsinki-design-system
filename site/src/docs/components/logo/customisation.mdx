## Customisation

<Notification type="alert" label="Logo customisation works differently!" className="siteNotification">
  Logo is an older HDS component and therefore its customisation works slightly differently. Instead of the <code>theme</code> property, you need to use the <code>style</code> property. This will be fixed in a future update.
</Notification>

### Customisation properties
In order to customise how the component looks, you can either use the `style` property or add the class containing your custom styles into the `className` property.

See all available theme variables in the table below.


### Customisation example
<Playground>

```jsx
<>
<Logo language="fi" size="small" style={{color: "var(--color-bus)", marginRight: 'var(--spacing-xs)'}} />
<Logo language="fi" size="medium" style={{color: "var(--color-tram)", marginRight: 'var(--spacing-xs)'}} />
<Logo language="fi" size="large" style={{color: "var(--color-metro)"}} />
</>
```

</Playground>
