## Customisation

<Notification type="alert" label="Logo customisation works differently!" className="siteNotification">
  Logo is an older HDS component and therefore its customisation works slightly differently. Instead of the 
  <code>theme</code> property, you need to use the <code>style</code> property. This will be fixed in a future update.
</Notification>

### Customisation properties

In Core version, you can either use the `style` or `class` attributes in the HTML to customise the component.

In React version, you can either use the `style` or `className` properties to customise the component.

### Customisation example

<Playground>

```jsx
<>
  <Logo language="fi" size="small" style={{ color: 'var(--color-bus)', marginRight: 'var(--spacing-xs)' }} />
  <Logo language="fi" size="medium" style={{ color: 'var(--color-tram)', marginRight: 'var(--spacing-xs)' }} />
  <Logo language="fi" size="large" style={{ color: 'var(--color-metro)' }} />
</>
```
