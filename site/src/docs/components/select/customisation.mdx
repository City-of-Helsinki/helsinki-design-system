## Customisation

### Customisation properties

You can use the `theme` property to customise the component. See all available theme variables in the table below.

| Theme variable                                      | Description                                                               |

| `--assistive-color`                                 | Text color of the assistive text.                                         |
| `--checkbox-background-default`                     | Background color of a checkbox in default, unselected state.              |
| `--checkbox-background-selected`                    | Background color of a checkbox in selected state.                         |
| `--checkbox-background-hover`                       | Background color of a checkbox when hovered.                              |

| `--checkbox-border-color-unselected-focus`          | Border color of a checkbox in unselected state when focused.              |
| `--checkbox-border-color-disabled`                  | Border color of a checkbox when disabled.                                 |
