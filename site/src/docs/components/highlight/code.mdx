### Packages

### Properties

| Property  | Description                     | Values                        | Default value |
| --------- | ------------------------------- | ----------------------------- | ------------- |
| `text`    | Heading for the Highlight/quote | -                             | -             |
| `variant` | Highlight variant               | `"small"` `"medium"` `"large"`| `"medium"`    | 
| `type`    | Type of the Highlight           | `"highlight"` `"quote"`       | `"highlight"` |
