## Customisation

### Customisation properties
You can either use the `style` or `class`Â attributes in the HTML to customise the component.

### Customisation example
<Playground scope={{ ErrorSummary }}>


```jsx
import { ErrorSummary } from 'hds-react';

() => (
 <ErrorSummary
  label="Error summary"
  style={{
    '--notification-background-color': 'lightgray',
    '--notification-border-color': 'black',
  }}
>
    <ul>
      <li>
        Error 1: <a href="#field1">Please enter your first name</a>
      </li>
      <li>
        Error 2: <a href="#field2">Please enter your last name</a>
      </li>
      <li>
        Error 3: <a href="#field3">Please enter a valid email address</a>
      </li>
    </ul>
  </ErrorSummary>
)
```
