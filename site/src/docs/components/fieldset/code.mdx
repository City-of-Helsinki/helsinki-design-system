### Properties

Note! You can find the full list of properties in the <Link openInNewTab openInNewTabAriaLabel="Opens in a new tab" href="/storybook/react/?path=/story/components-dropdowns-select--default">React Storybook.</Link>

| Property              | Description                                            | Values    | Default value |
| --------------------- | ------------------------------------------------------ | --------- | ------------- |
| `heading`             | Heading of the fieldset element.                       | `string`  | -             |
| `border`              | If set to true, a border is drawn around the fieldset. | `boolean` | false         |
| `tooltipLabel`        | The aria-label text for the tooltip.                   | `string`  | -             |
| `tooltipButtonLabel`  | The aria-label text for the tooltip trigger button.    | `string`  | -             |
| `tooltipText`         | The text content of the tooltip.                       | `string`  | -             |
