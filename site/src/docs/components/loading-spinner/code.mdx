### Properties
Note! You can find the full list of properties in the <Link openInNewTab openInNewTabAriaLabel="Opens in a new tab" href="/storybook/react/?path=/story/components-loadingspinner--default">React Storybook.</Link>

| Property              | Description                                              | Values    | Default value               |
| --------------------- | -------------------------------------------------------- | --------- | --------------------------- |
| `multicolor`          | If set to true, the multicolour variant is used.         | `boolean` | false                       |
| `small`               | If set to true, the small variant is used.               | `boolean` | false                       |
| `loadingText`         | Text to show for screen readers when spinner is visible. | `string`  | "Page is loading"          |
| `loadingFinishedText` | Text to show for screen readers when spinner is removed. | `string`  | "Page has finished loading" |
