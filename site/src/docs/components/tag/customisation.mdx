## Customisation

### Customisation properties
In Core version, you can either use the `style` or `class` attributes in the HTML to customise the component.

In React version, you can use the `theme` property to customise the component.

See all available theme variables in the table below.

| Theme variable                        | Description                                 |
| ------------------------------------- | ------------------------------------------- |
| '--tag-action-border-color'           | Colour of the border of Tags with onClick or onDelete actions |
| '--tag-background-color'              | Colour of the tag background.               |
| '--tag-color'                         | Colour of elements inside the tag.          |
| '--tag-focus-outline-color'           | Colour of the tag focus border.             |
| '--tag-hover-background-color'        | Colour of the tag background when hovered upon. |

### Customisation example
<Playground scope={{ Tag }}>


```jsx
import { Tag } from 'hds-react';

() => (
<Tag
  aria-label="run custom action"
  onClick={function noRefCheck(){}}
  theme={{
    '--tag-background-color': 'var(--color-copper-dark)',
    '--tag-hover-background-color': 'var(--color-tram-dark)',
    '--tag-color': 'var(--color-white)',
    '--tag-focus-outline-color': 'var(--color-metro)',
  }}
>
  Action
</Tag>)
```

```html
<div
  role="button"
  class="hds-tag hds-tag--action"
  style="
    --tag-background-color: var(--color-copper-dark);
    --tag-hover-background-color: var(--color-tram-dark);
    --tag-color: var(--color-white);
    --tag-focus-outline-color: var(--color-metro);
  "
  tabindex="0"
  onclick={}
  aria-label="run custom action"
>
  <span class="hds-tag__label">Action</span>
</div>
```
