### Properties
Note! You can find the full list of properties in the <Link openInNewTab openInNewTabAriaLabel="Opens in a new tab" href="/storybook/react/?path=/docs/components-tag--default">React Storybook.</Link>

| Property                | Description                                                                                                                    | Values              | Default value |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------ | ------------------- | ------------- |
| `aria-label`            | The aria-label for the interactive Tags (with onClick, onDelete or href -properties)                                           | `string`            | -             |
| `href`                  | Url to go to when clicked, defines the Tag as a link                                                                           | `string`            | -             |
| `iconLeft`              | Icon placed on the left side of the label.                                                                                     | `ReactNode`         | -             |
| `iconRight`             | Icon placed on the right side of the label.                                                                                    | `ReactNode`         | -             |
| `onClick`               | Callback function fired when the tag is clicked. If set, the tag will be clickable.                                            | `function`          | -             |
| `onDelete`              | Callback function fired when the tag is clicked. If set, the tag will be clickable and a delete icon will be shown.            | `function`          | -             |
| `size`                  | Sets the size of the Tag.                                                                                                      | `"s"` `"l"`         | `"s"`         |
| `target`                | The target attribute specifies where to open the linked document.                                                              | `"_blank"` `"_self"` `"_parent"` `"_top"` `"framename"` | `"_self"`         |
