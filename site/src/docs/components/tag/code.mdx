### Properties
Note! You can find the full list of properties in the <Link openInNewTab openInNewTabAriaLabel="Opens in a new tab" href="/storybook/react/?path=/docs/components-tag--default">React Storybook.</Link>

| Property                | Description                                                                                                                    | Values              | Default value |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------ | ------------------- | ------------- |
| `deleteButtonAriaLabel` | The aria-label for the delete button                                                                                           | `string`            | -             |
| `deleteButtonProps`     | Props that will be passed to the delete button `button` element.                                                               | `properties`        | -             |
| `labelProps`            | Props that will be passed to the label `span` element.                                                                         | `properties`        | -             |
| `onClick`               | Callback function fired when the tag is clicked. If set, the tag will be clickable.                                            | `function`          | -             |
| `onDelete`              | Callback function fired when the delete icon is clicked. If set, a delete button will be shown.                                | `function`          | -             |
| `role`                  | Sets the role of the tag when it's clickable. Uses 'link' by default.                                                          | `"button"` `"link"` | `"link"`      |
| `srOnlyLabel`           | Label that is only visible to screen readers. Can be used to to give screen reader users additional information about the tag. | `string`            | -             |
