# Releases

## Coming up

Next quarter we are working on these larger features and fixes. Smaller feature updates will be coming up regularly.
## Version 4.0
### Cookie consent
- design updates to simplify the user experience
- core styles for the banner and page
- bug fixes
- investigations on web components and js hooks to provide the cookie management to different platforms

### Dropdown refactoring
- design updates to make it clearer
- removing outdated downshift-library
- improving accessibility
- bug fixes
- refactor code

### Tag improvements
- related to dropdown refactoring
- accessibility improvements
- code clarification

### Other updates
- unifiying line heights with hel.fi and HDS and improving readability especially with larger titles
- Button fixes: better maintainability of the code, data-attributes and better support for right-to-left positioning
- Improve contrast for black-60
- unified size-property across components
- removing default id value

## Version 5.0 
- React update to 18.x
- this is done separataly from component updates to enable better testing and easier migration
