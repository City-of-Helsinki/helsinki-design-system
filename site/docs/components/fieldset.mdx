<LargeParagraph>Fieldset allows grouping of related form inputs.</LargeParagraph>

## Principles

- Fieldset component should be used to group related form inputs.
- A heading must be given to the Fieldset. The heading should be clear and describe the input group.
- **HDS Fieldset does not style the content in any way.** You may have to add space between the inputs to distinguish them from each other.

## Usage & variants

### Default

By default, Fieldset component does not have a border.

#### React code example:

<Playground>
  <Fieldset heading="Applicant information" style={{ maxWidth: '400px' }}>
    <TextInput id="first-name" name="first-name" label="First name" />
    <TextInput id="last-name" name="last-name" label="Last name" style={{ marginTop: '12px' }} />
    <TextInput
      id="social-security-number"
      name="social-security-number"
      label="Social security number"
      placeholder="Eg. 111299-1234"
      style={{ marginTop: '12px' }}
    />
  </Fieldset>
</Playground>

#### Core code example:

```html
<fieldset class="hds-fieldset" style="max-width: 400px">
  <legend class="hds-fieldset-legend">Applicant information</legend>
  <div class="hds-text-input">
    First name *
    <div class="hds-text-input__input-wrapper">
      <input id="first-name" name="first-name" class="hds-text-input__input" type="text" />
    </div>
  </div>
  <div class="hds-text-input" style="margin-top: 12px">
    Last name *
    <div class="hds-text-input__input-wrapper">
      <input id="last-name" name="last-name" class="hds-text-input__input" type="text" />
    </div>
  </div>
  <div class="hds-text-input" style="margin-top: 12px">
    Last name *
    <div class="hds-text-input__input-wrapper">
      <input id="last-name" name="last-name" class="hds-text-input__input" type="text" />
    </div>
  </div>
</fieldset>
```
